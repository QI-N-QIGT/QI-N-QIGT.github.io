人类社会与人性

发表于 2019-08-28 | 分类于 社会、国家、人
字数统计: 1k | 阅读时长 ≈ 3

风从山谷里吹过，吹过明丽的花，纯净的草，也拂过黑色的土，黄色的沙。

客观的世界绝非是完美的，甚至说离完美相去甚远。如果能看到这个世界的更多的客观真相而不增加对其失望情绪，那么这个人不是看不懂，就是有极强功力封锁情绪以至于不外露的超人。

世界并不总是能够主持正义，人与人又如何能够获得平等。我很明确的知道平等并不绝对是件好事情，但看起来不平等总让人觉得心里发痛，像是一种不正义，使人渴望平等。

不平等能让人类社会更有效的运转，更加的有活力有效率，更加的有前进的朝气。与此同时，不平等会造成人类之间可怕的相互欺凌与难以忽视的加大差距的趋势。

从整个人类整体的角度来看，不平等有其存在的巨大价值，正如前面所说。但当镜头聚焦到一个个的人，以及其身后的一个个的故事之时，不平等所引发的种种效应会让人的同情心无限生发，进而放大至对不平等的深恶痛绝。

人类社会的发展进程显示，那些要求平等的人往往是在当时不平等状态下的劣势者。我想，人类社会极难实现完全平等，或者说我对其抱有悲观态度。即使整个人类社会都由那些要求平等的人类个体组成，我想，最终所组成的那个社会也大概率的是不平等的，回过头看，也不过就是不平等社会的一次洗牌。

不平等并不能意味着不正义，平等也绝不能简单的认为是正义的象征。无论如何，都不可能把正义与平等，不正义与不平等这些挂上钩，从经验上来判断，我以为平等与否与正义与否没有一个确定不变的关系。正不正义是要放在具体事件中看待，你的判断极大的受你观察的角度以及你的讲述人的主观情绪所影响。这样，你对这个世界正义与否的判断就不是在客观世界进行的，而是在主观世界里进行的，这无论如何都极难公正。

社会的不平等与平等根本，来源于人的本性而非人类社会的制度。社会特性，是经过许多人性的交织杂糅变化而后形成的一种属于整体的性质。就像是一个自变量，经过一个函数得到了一个因变量。因变量变化根本的是由于自变量的变化，因变量变化的特性则取决于自变量变化的特性与函数的特性。所以，看待社会纷繁复杂的诡谲现象，都能在难以理解的个人的人性上找到依据。人性的灵活以及其极为复杂的特点，很难用一套固定的制度，即连接人与人，规定人之间的关系的一些人造产物所束缚或言规定。

如此来看，研究社会更本质的是在研究人，管理社会也更多的是在管理人。我想，当大部分的人能够靠着什么东西（理性或是其他什么东西）长期并且在自己不痛苦的情况下管理自己的天性或是某些人掌握了长期且不令人痛苦的管理人性的方法并把它固化为制度，也许这样就能够使得绝大部分的社会问题得以解决。但如若真的有人能够做到如此，我想那时的社会是天堂还是地狱，却实在是不好说的事情。

​ –秦琦
